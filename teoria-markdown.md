## Bloques de Código

```sql
INSERT INTO tabla (campo_1, campo_2, ..., campo_n)
VALUES (valor_1, valor2, ..., valor_n);

INSERT INTO tabla
SET campo_1 = 'valor_1', campo_2 = 'valor_2', ..., campo_n = valor_n;
```

```js
function sumar(a, b) {
  return a + b;
}
```

```html
<ul>
  <li>Primavera</li>
  <li>Verano</li>
  <li>Otoño</li>
  <li>Invierno</li>
</ul>
<form>
  <input type="text" />
  <input type="button" />
</form>
```
